# 白屏快速排查手册

## 一、最常见原因（按优先级）

### 1. 浏览器缓存了无效登录凭证
**现象**：页面全白，F12 Console 无报错

**解决**：
- 打开 `http://localhost:5173`
- 按 `F12` → `Application` → `Local Storage` → 右键 `Clear`
- 刷新页面 → 应该看到登录页

**自动修复**：新版本启动时会自动向后端验证凭证，无效凭证自动清除。

### 2. 后端服务未启动
**现象**：页面显示登录框但无法注册/登录，Network 显示红色

**检查**：终端运行 `curl http://localhost:3001/api/config`，应返回 JSON。

**解决**：双击 `start.bat` 重新启动。

### 3. 端口被占用
**现象**：启动时报 `EADDRINUSE`

**解决**：
```bash
netstat -ano | findstr :3001
taskkill /F /PID <PID号>
```
然后重新启动。

### 4. 数据库文件损坏
**现象**：API 返回 500 错误

**解决**：双击 `backup-restore.bat`，选择 `[1] 恢复最近备份`

## 二、手动排查步骤

```
1. 开终端，运行: curl http://localhost:3001/api/config
   → 有 JSON 返回 = 后端正常
   → 无响应 = 后端没启动

2. 浏览器 F12 → Console 标签
   → 红字报错 = 前端代码问题
   → 无报错 = 认证/缓存问题

3. F12 → Application → Local Storage → 清空 → F5 刷新
   → 看到登录页 = 缓存问题（已自动修复）
   → 仍然白屏 = 继续下一步

4. 双击 check-tunnel.bat 检查服务状态
```

## 三、极限恢复

```
1. 双击 backup-restore.bat → 选 [1] 恢复数据库
2. 重启电脑
3. 双击 start.bat 启动
4. 浏览器无痕窗口打开 http://localhost:5173
```
