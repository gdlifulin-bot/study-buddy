# 项目修改指南

快速定位修改入口，方便后续迭代。

## 修改背景图

| 修改项 | 文件位置 | 说明 |
|--------|----------|------|
| 默认背景色/渐变 | [HomePage.jsx](client/src/features/home/HomePage.jsx) 第 30 行 | 修改 `bg-gradient-to-br` 相关 class |
| 上传自定义背景 | [BackgroundPicker.jsx](client/src/features/home/BackgroundPicker.jsx) | 上传后图片存到 `server/uploads/` |
| 背景数据存储 | [server/routes/config.js](server/routes/config.js) | config.json 中的 `backgroundType` 字段 |

## 调整 UI 配色/样式

| 修改项 | 文件位置 | 说明 |
|--------|----------|------|
| 全局配色 | [tailwind.config.js](client/tailwind.config.js) | `colors.apple` 对象 |
| 业务常量 | [constants.js](client/src/config/constants.js) | 科目列表、用户信息 |
| 全局 CSS | [index.css](client/src/index.css) | 字体、滚动条、毛玻璃效果 |

## 新增功能模块

1. 在 [features/](client/src/features/) 下新建目录
2. 在 [App.jsx](client/src/App.jsx) 添加路由
3. 如需后端 API，在 [server/routes/](server/routes/) 新建路由文件
4. 在 [server/index.js](server/index.js) 注册路由

## 新增考研科目

编辑 [constants.js](client/src/config/constants.js) 的 `SUBJECTS` 数组，所有引用此数组的组件自动适配。

## 接入 AI 生成计划

编辑 [AIPlanGenerator.jsx](client/src/features/planner/AIPlanGenerator.jsx) 中的 `generatePlanAI` 函数（第 18 行），替换占位实现为真实 API 调用。

## 切换数据存储（JSON → 数据库）

编辑 [server/index.js](server/index.js) 中的 `readJSON` 和 `writeJSON` 函数，替换为数据库读写逻辑。所有路由代码无需改动。

## 状态管理升级（Context → Redux）

Context 文件位于 [contexts/](client/src/contexts/)，每个 Context 薄封装，可逐个替换为 Redux slice。组件无需修改。

## 部署到生产环境

```bash
cd client && npm run build   # 构建前端
cd .. && set NODE_ENV=production && cd server && node index.js  # 启动生产服务
```

访问 `http://localhost:3001` 即可使用。
